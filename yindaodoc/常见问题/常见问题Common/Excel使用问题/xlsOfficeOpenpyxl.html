xls文件，强制修改文件后缀为xlsx，使用office或openpyxl打开文件会报错

问题描述

有些情形，被下载文件为 03 格式的 excel 文件，如 123.xls，下载文件时，用户自定义保存文件名的过程中，将文件后缀改为 07 格式，如 456.xlsx，此时用 office 或 openpyxl 驱动打开该 Excel，会报错：the file format or file extension is not valid。

解决方案

使用 WPS 的驱动方式打开 Excel。

其他解决方案

另存为时，不要改变文件后缀。

问题没有解决？去社区提问 all right reserved，powered by Gitbook
